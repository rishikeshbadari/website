# Fractals

Welcome to my collection of fractal visualizations! This project explores the fascinating world of fractals through interactive implementations.

## What are Fractals?

Fractals are infinitely complex patterns that are self-similar across different scales. They are created by repeating a simple process over and over in an ongoing feedback loop. Driven by recursion, fractals are images of dynamic systems – the pictures of Chaos.

## Available Fractal Implementations

This project contains several different fractal implementations, each showcasing different mathematical concepts and visual patterns:

### 🌿 Barnsley Fern
An iterated function system that creates a beautiful fern-like pattern using probability-driven transformations.

### 🌀 Mandelbrot Set
The most famous fractal, generated by iterating a simple complex function and testing for divergence.

### 🎯 Julia Sets
Complex fractals that are closely related to the Mandelbrot set, each with its own unique character based on a complex parameter.
